# How to use

Just clone this project:
```bash
git clone https://github.com/Naxn1a/workspace-bn.git
```
## Local setup
### Install the dependencies
```bash
npm install
```

### Setup `.env.local` file

```bash
REST_API={backend_url}
```

### `Develop`
```bash
npm run dev
# or
yarn dev
```

### `Start`
```bash
npm start
# or
yarn start
```

### `Build`
```bash
npm build
# or
yarn build
```

### Goodluck!
